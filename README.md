# 🛡️ OWASP Juice Shop – Risk Assessment & ISO 27001 Mapping

This project demonstrates a practical application of risk management and control mapping based on a deliberately vulnerable application: OWASP Juice Shop.

## 🎯 Objectives
- Identify and analyze security risks
- Perform basic threat modeling
- Map findings to ISO/IEC 27001 controls
- Propose remediation strategies

## 🧠 Methodology
- Qualitative risk analysis (Likelihood × Impact)
- OWASP Top 10 as reference
- Asset-based approach

## 📊 Deliverables
- Risk Register
- Threat Model
- ISO 27001 Mapping
- Gap Analysis
- Recommendations
